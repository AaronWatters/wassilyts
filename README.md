# wassilyts
Infrastructure for building scientific diagrams in 2d and 3d using HTML5 canvases with animations and interactivity.


- `npm run build` — Compiles and bundles the library using Vite for production output.
- `npm run test` — Runs the unit test suite using Vitest.
- `npm run dev` — Starts the Vite development server for local development.
- `npm run test:e2e` — Builds the project and then runs end-to-end tests using Playwright.
- `npm run test:coverage` — Runs the unit test suite with code coverage reporting via Vitest.

## Update all dependencies

```bash
rm -r package-lock.json node_modules
npm install
```
